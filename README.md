Практическое задание по дисциплине «Введение в HTML»
## Описание
В рамках задания выполнены 4 задачи:
1. Базовая HTML-страница с форматированием текста.
2. Персональная веб-страница (заголовки, списки, таблица, медиа, ссылки).
3. Форма обратной связи с валидацией.
4. Страница с семантической разметкой HTML5 (header, nav, main, section, footer).
